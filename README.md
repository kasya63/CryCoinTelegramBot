# CryCoin Telegram Bot

Telegram-бот для учёта «времени плача» (CryCoin): **4 часа в неделю** на человека, кнопки **Старт** / **Стоп**, автоостановка при исчерпании лимита и сброс по **понедельникам 00:00 (Астана, `Asia/Almaty`)**.

## Быстрый старт

### 1. Создайте бота в BotFather

1. Откройте [@BotFather](https://t.me/BotFather).
2. Отправьте `/newbot`.
3. Укажите имя (например, `CryCoin Timer`) и username (например, `MyCryCoinBot`).
4. Скопируйте **токен** — он понадобится как `BOT_TOKEN`.

**Рекомендуется:**

- `/setdescription` — «4 часа плача в неделю»
- `/setcommands`:
  ```
  start - Запуск и кнопки
  status - Остаток времени
  help - Правила
  ```
- `/setprivacy` → **Disable** — чтобы бот видел сообщения в группе

### 2. Настройка и запуск локально

```bash
cd ~/crycoin-bot
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
# Отредактируйте .env и вставьте BOT_TOKEN
python -m bot.main
```

Пока процесс запущен — бот работает.

### 3. Добавьте бота в чат

- **Личка:** нажмите Start у бота.
- **Группа:** добавьте бота участником и дайте право писать сообщения.

## Переменные окружения

| Переменная | Описание | По умолчанию |
|------------|----------|--------------|
| `BOT_TOKEN` | Токен от BotFather | обязательно |
| `TIMEZONE` | Часовой пояс сброса | `Asia/Almaty` |
| `WEEKLY_LIMIT_SECONDS` | Недельный лимит в секундах | `14400` (4 ч) |
| `DB_PATH` | Путь к SQLite-базе | `crycoin.db` |
| `TICK_INTERVAL_SECONDS` | Интервал проверки авто-стопа | `30` |

## Команды бота

- `/start` — приветствие и кнопки Старт / Стоп
- `/status` — сколько времени осталось на неделю
- `/help` — правила CryCoin

## Правила

- У каждого пользователя свой лимит 4 часа в неделю.
- Одна активная сессия на человека.
- Стоп может нажать только тот, кто нажал Старт.
- При исчерпании лимита бот автоматически останавливает таймер и **не отвечает** этому пользователю до понедельника 00:00.
- Неиспользованное время **не переносится** — каждую неделю снова 4 часа.

## Деплой на Railway (24/7)

1. Зарегистрируйтесь на [railway.app](https://railway.app).
2. Создайте проект → Deploy from GitHub (или загрузите репозиторий).
3. В **Variables** добавьте `BOT_TOKEN` (и при необходимости остальные переменные).
4. Railway соберёт образ по `Dockerfile` и запустит бота.

Для постоянной работы на Railway добавьте volume или внешнее хранилище, если нужна сохранность `crycoin.db` между перезапусками (на бесплатном плане данные могут сбрасываться при редеплое).
